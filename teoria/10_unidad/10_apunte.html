Práctica: Introducción a Google Colab y Python

Esta práctica está diseñada para estudiantes que ya tuvieron un primer acercamiento a R y ahora comenzarán a trabajar con Google Colab y Python.

Google Colab permite escribir texto, ejecutar código y visualizar resultados dentro de un mismo documento llamado notebook. Este formato es muy útil para aprender programación, analizar datos y documentar procedimientos paso a paso.

Objetivos de la práctica

Al finalizar esta actividad, podrás:

  • Identificar las partes principales de un notebook de Google Colab.
  • Ejecutar celdas de código en Python.
  • Crear variables y realizar operaciones básicas.
  • Usar listas y ciclos.
  • Crear y explorar un DataFrame con pandas.
  • Elaborar gráficas simples con matplotlib.
  • Comparar algunos conceptos vistos en R con su equivalente en Python.

Recomendaciones antes de iniciar

  1. Lee cada explicación antes de ejecutar el código.
  2. Ejecuta las celdas en orden.
  3. Modifica algunos valores para observar qué cambia.
  4. Responde las preguntas en una celda de texto debajo de cada sección.
  5. Guarda tu archivo con tu nombre.

Parte 1 — Primeros pasos en Google Colab

Google Colab funciona como un cuaderno interactivo. En él podemos combinar:

  • Celdas de texto, donde escribimos explicaciones, títulos, instrucciones o respuestas.
  • Celdas de código, donde escribimos instrucciones en Python y las ejecutamos.

A diferencia de un archivo tradicional de código, un notebook permite ver los resultados inmediatamente debajo de cada celda. Esto facilita la exploración y el aprendizaje.

Instrucciones

  1. Abre Google Colab.
  2. Crea un nuevo notebook.
  3. Cambia el nombre del archivo a:
Practica_Colab_NombreApellido.ipynb
  1. Ejecuta las celdas usando el botón de reproducción o con Shift + Enter.

Parte 2 — Primer programa en Python

En programación, es común iniciar con un programa sencillo que muestre el mensaje Hola mundo.

En Python usamos la función print() para mostrar información en pantalla. Una función es una instrucción ya definida que realiza una tarea específica.

En este caso, print() recibe un texto y lo muestra como salida.

print("Hola mundo")
Hola mundo
print("Hola mundo")

print("Buenos días")
Hola mundo
Buenos días

Variables

Una variable es un espacio donde guardamos información para usarla después.

En Python, para crear una variable usamos el signo =. Por ejemplo:

nombre = "Ana"

Esto significa que estamos guardando el texto "Ana" dentro de la variable llamada nombre.

A diferencia de R, donde muchas veces usamos <-, en Python normalmente usamos = para asignar valores.

nombre = "Ana"
edad = 20

print(nombre)
print(edad)
Ana
20
nombre = "Linda"
Edad = 33
print(nombre)
print(Edad)
Linda
33

Parte 3 — Tipos de datos y operaciones básicas

Python puede trabajar con diferentes tipos de datos. Algunos de los más comunes son:

  • int: números enteros, por ejemplo 10.
  • float: números decimales, por ejemplo 3.14.
  • str: texto o cadenas de caracteres, por ejemplo "Hola".
  • bool: valores lógicos, es decir True o False.

También podemos realizar operaciones matemáticas usando operadores.

a = 10
b = 3

print("Suma:", a + b)
print("Resta:", a - b)
print("Multiplicación:", a * b)
print("División:", a / b)
print("Potencia:", a ** b)
Suma: 13
Resta: 7
Multiplicación: 30
División: 3.3333333333333335
Potencia: 1000

Explicación de los operadores

Operador Significado Ejemplo
+ Suma a + b
- Resta a - b
* Multiplicación a * b
/ División a / b
** Potencia a ** b

En Python, la potencia se escribe con doble asterisco **. En R también se puede usar ^, pero en Python ^ no se utiliza para elevar potencias.

Parte 4 — Listas y ciclos

Una lista en Python permite guardar varios valores dentro de una sola variable.

En R trabajamos con vectores, por ejemplo:

numeros <- c(1, 2, 3, 4, 5)

En Python usamos corchetes []:

numeros = [1, 2, 3, 4, 5]

Una diferencia importante es que Python empieza a contar posiciones desde 0, no desde 1.

numeros = [1, 2, 3, 4, 5]

print(numeros)
print("Primer elemento:", numeros[0])
print("Último elemento:", numeros[-1])
[1, 2, 3, 4, 5]
Primer elemento: 1
Último elemento: 5

Ciclo for

Un ciclo permite repetir instrucciones.

El ciclo for se usa cuando queremos recorrer una secuencia de elementos, como una lista.

En el siguiente ejemplo, Python toma cada elemento de la lista numeros y lo guarda temporalmente en la variable n.

for n in numeros:
    print(n)
1
2
3
4
5

Parte 5 — Introducción a pandas

pandas es una biblioteca de Python utilizada para trabajar con datos en forma de tablas.

En R, una estructura muy común es el data.frame. En Python, con pandas, usamos el DataFrame.

Un DataFrame tiene filas y columnas, similar a una hoja de cálculo.

Para usar pandas, primero debemos importarlo.

import pandas as pd

Crear un DataFrame

Primero crearemos un diccionario de Python. Un diccionario guarda información en pares de clave y valor.

En este caso:

  • Las claves serán los nombres de las columnas.
  • Los valores serán listas con los datos de cada columna.
datos = {
    "Nombre": ["Ana", "Luis", "Carlos", "María"],
    "Edad": [20, 21, 19, 22],
    "Calificacion": [90, 85, 95, 88]
}
print(datos)
df = pd.DataFrame(datos)

Calcular estadísticas simples

También podemos calcular valores como el promedio, el valor máximo y el valor mínimo de una columna.

promedio_calificaciones = df["Calificacion"].mean()

Parte 6 — Visualización de datos con matplotlib

Las gráficas nos ayudan a entender mejor los datos.

En Python podemos usar la biblioteca matplotlib para crear visualizaciones básicas.

Primero importamos el módulo pyplot, que contiene funciones para crear gráficas.

import matplotlib.pyplot as plt

Gráfica de barras

Una gráfica de barras es útil para comparar valores entre categorías.

En este ejemplo, compararemos las calificaciones de cada estudiante.

plt.bar(df["Nombre"], df["Calificacion"])

plt.title("Calificaciones de estudiantes")
plt.xlabel("Estudiantes")
plt.ylabel("Calificación")

plt.show()

Gráfica de dispersión

Una gráfica de dispersión muestra la relación entre dos variables numéricas.

En este caso, observaremos la relación entre la edad y la calificación.

plt.scatter(df["Edad"], df["Calificacion"])

plt.title("Edad y calificación")
plt.xlabel("Edad")
plt.ylabel("Calificación")

plt.show()

Parte 8 — Actividad extra: subir un archivo CSV

Google Colab permite subir archivos desde tu computadora.

Un archivo CSV es un archivo de texto que guarda datos en forma de tabla, separados por comas. Es muy usado para trabajar con bases de datos pequeñas o medianas.

Para subir un archivo en Colab podemos usar files.upload().

Nota: esta celda funciona correctamente en Google Colab. En otros entornos puede no ejecutarse.

from google.colab import files

uploaded = files.upload()
