Introducción

En esta sesión aprenderemos los principios básicos de manipulación de datos en R utilizando el paquete dplyr, que forma parte del ecosistema tidyverse.

  • Tidyverse para análisis de datos

El objetivo principal es que el estudiantado:

  • Comprenda la lógica de la manipulación de datos tipo tidy

  • Aprenda a leer y escribir código más claro y reproducible

  • Se familiarice con funciones esenciales para el análisis de datos

  • Trabajaremos con un conjunto de datos real: penguins, ampliamente utilizado con fines educativos.

Paquetes necesarios

En R, gran parte de la funcionalidad se encuentra en paquetes. Para esta clase utilizaremos:

El conjunto de datos penguins

El conjunto de datos penguins contiene información morfológica de tres especies de pingüinos observados en distintas islas de la Antártida.

Algunas variables incluyen:

  • especie
  • isla
  • longitud y profundidad del pico
  • longitud de la aleta
  • masa corporal
  • sexo

El operador pipe %>%

Una de las ideas centrales de dplyr es el uso del operador pipe (%>%).

  • %>% o |>
  • Atajos en el teclado
  • Windows / Linux: Ctrl + Shift +
  • MmacOS: Cmd + Shift + M

Este operador permite encadenar funciones, haciendo que el código:

  • sea más legible
  • siga una lógica paso a paso
  • se parezca al lenguaje natural

Puede leerse como “y luego”.

mutate(): Crear nuevas variables

La función mutate() se utiliza para crear o transformar variables.

Ejemplo: convertir la masa corporal de gramos a libras.

penguins %>%
  mutate(body_weight_pounds = body_mass_g / 453.59237) %>%
  select(body_mass_g, body_weight_pounds) %>%
  head()
# A tibble: 6 × 2
  body_mass_g body_weight_pounds
        <int>              <dbl>
1        3750               8.27
2        3800               8.38
3        3250               7.17
4          NA              NA   
5        3450               7.61
6        3650               8.05

summarize(): Resumir datos

summarize() permite calcular estadísticas descriptivas.

penguins %>%
  summarize(avg_body_mass = mean(body_mass_g, na.rm = TRUE))
# A tibble: 1 × 1
  avg_body_mass
          <dbl>
1         4202.

El argumento na.rm = TRUE indica que los valores faltantes (NA) deben ignorarse.
